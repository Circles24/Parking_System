# Parking_System
Simple java parking management system

## Installation

in case you don't have git

 `sudo apt-get install git`
 
Clone the repo
 
 `git clone https://github.com/Circles24/Parking_System.git`
 
Open the directory

`cd Parking_System`

## Usage

### Building the application

`./build.sh`

### Running the server

`./server.sh`

### Running the client

`./client.sh`
